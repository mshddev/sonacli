# Overview

This is `sonacli`, a small CLI for consuming SonarQube analysis reports.

The project currently targets:

- SonarQube Community Build v25.x+ only (with MQR Mode)
- Linux and macOS only

Out of scope:

- Windows support
- SonarCloud support
- SonarQube Server / Data Center support
- Non-Web API inputs

- Follow `CONTRIBUTING.md` for repository workflow, local environment setup, and testing policy.
- Follow `tests/README.md` for end-to-end test execution and reporting.
